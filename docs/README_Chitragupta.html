🔗 API Endpoints

Authentication (Layer 1 & 2)

MethodEndpointDescription
POST/api/auth/registerRegister with OTP verification
POST/api/auth/verify-otpVerify 6-digit OTP code
POST/api/auth/loginLogin (returns JWT pair)
POST/api/auth/refreshRotate JWT tokens
POST/api/auth/totp/setupGenerate TOTP QR for 2FA

Admin Security (Layer 4 & 5)

MethodEndpointDescription
GET/api/admin/logsFetch immutable audit logs
GET/api/admin/logs/export/csvExport logs as CSV
GET/api/admin/logs/export/pdfExport logs as PDF
POST/api/admin/scanner/runTrigger vulnerability scan
GET/api/admin/scanner/reportsFetch scan reports
